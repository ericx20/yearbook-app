# yearbook-app
Virtual yearbook signing

A web app that lets you sign yearbooks online!

The front end is a fully functional drawing app that supports customizable brush, undo/redo, and PNG image export!

The Node.js backend is located on the [yearbook-server repository.](https://github.com/ericx20/yearbook-server) As a REST API, it provides functionality for CRUD operations with yearbooks, as well as a database for storing the pages. 

Currently, I'm finishing up the front end so it can communicate with the server. Once that's done, I plan to showcase a live demo!
