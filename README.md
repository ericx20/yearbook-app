# yearbook-app
Virtual yearbook signing

A web app that lets you sign yearbooks online!

Currently, this repository only contains the frontend component. It's a fully functional drawing app that supports customizable brush, undo/redo, and PNG image export!

The Node backend is located on the [yearbook-server repository](https://github.com/ericx20/yearbook-server), and for now it's a REST API. Very soon, both frontend and backend will be able to communicate with each other and deliver the full experience.
